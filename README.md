# TheCloudTheory Topaz

## How do I install these formulae?

`brew install thecloudtheory/topaz/<formula>`

Or `brew tap thecloudtheory/topaz` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "thecloudtheory/topaz"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
